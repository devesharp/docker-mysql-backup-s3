## Usage

```shell
docker run \
    -e "MYSQL_HOST=value" \
    -e "MYSQL_USER=value" \
    -e "MYSQL_PASSWORD=value" \
    -e "MYSQL_DB=value" \
    -e "AWS_BUCKET=value" \
    -e "AWS_REGION=value" \
    -e "AWS_ACCESS_KEY=value" \
    -e "AWS_SECRET_KEY=value" \
    -e "PASSWORD_ZIP=value" \
    devesharp/mysql-backup-s3
```
